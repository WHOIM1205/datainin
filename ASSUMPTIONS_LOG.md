# Assumptions & Decisions Log

Single, authoritative record of what we **know** (verified facts) and what we **decided**
(assumptions/choices) across the project. Governance rules:

1. **Frozen facts are immutable** unless the *raw data itself* proves otherwise. They must
   never be silently contradicted by a later module.
2. **Every cleaning / feature / analysis decision must cite the relevant frozen fact (F-id).**
3. **Changing an earlier assumption requires a CHANGE entry** — what changed, why, and the
   evidence — appended below, never an in-place edit that hides the history.
4. This log is updated at the end of **every** module.
5. **Materiality bar (keep this log lightweight).** Only record a decision if it materially
   changes **cleaning, feature engineering, statistics, or business conclusions**. Do **not**
   log obvious implementation details, library choices, or anything a reader could infer from
   the code. If removing an entry wouldn't change how results are interpreted, it doesn't belong.
6. **EDA is frozen (Module 4).** The descriptive picture — distributions, balances, concentration,
   sampling — is settled in `outputs/eda_summary.md` and the E-ids in Part A2. EDA answers *"what
   does the data look like?"*; every later module answers *"what does it mean?"*. **Do not recompute
   or re-chart descriptive statistics** — cite the E-id / `eda_summary.md` instead. Recomputing an
   already-frozen descriptive result is duplicated work and is disallowed.

## Part A2 — Frozen EDA Findings (Module 4)

Canonical descriptive results. Evidence: `outputs/eda_summary.md` + `outputs/figures/`.
**Immutable and not to be recomputed** — later modules cite the E-id (governance rule 6).

| ID | Finding | Consequence for later modules | Source |
|----|---------|-------------------------------|--------|
| E1 | **Realized PnL is extremely heavy-tailed**: skew 21.6, excess kurtosis 3,163; mean 98.6 ≫ median 6.06; win rate 83.2% on 104,408 PnL-bearing (closing) trades | Use **non-parametric tests + effect sizes**, medians not means; winsorize for summaries | V2 |
| E2 | **Severe temporal concentration**: 0.2% of trades in 2023, **92.2% in the final 6 months** (Nov-2024→Apr-2025); 217-day zero-activity gap; 64.9% day-coverage | Effective window is late-2024→2025; a regime effect may proxy *when* trading happened → window/robustness decision (**D10**) | V6 |
| E3 | **Account concentration**: 32 accounts, Gini 0.57, top account 19%, top-5 50.5% of trades | Pooling violates independence → **per-account paired / cluster-aware designs** (D9) | V4 |
| E4 | **In-window sentiment balance (Greed-tilted)**: Greed 411 / Neutral 177 / Fear 144 days; mean index 58.2 | Adequate but imbalanced groups → tests robust to unequal n; Neutral kept (D5) | V1 |
| E5 | **Directional baseline**: Long 46.7% / Short 35.9% / Spot 17.4% / Edge 0.1%; perp long/short ratio 1.30 | The **null** for the later sentiment × direction chi-square; spot/edge excluded from directional tests | V5 |
| E6 | **Trade size is heavy-tailed**: 0.01 → 3.92M USD; median perp 649 vs spot 393 USD | Compare **medians / log-sizes**, not means; enables size-tier segmentation | V3 |
